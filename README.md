# 音乐创作小程序 - Mofei Music

🎵 人人都可以创作音乐的小程序

## 技术栈
- 前端：微信原生小程序 + Vant Weapp + TypeScript
- 后端：FastAPI + SQLite + Python 3.11
- 音乐 API：MiniMax

## 功能
- ✅ 示例引导 - 展示专业音乐示例，用户可一键使用
- ✅ 音乐创作 - 输入需求，一键生成音乐
- ✅ 在线播放 - 生成后直接在页面上播放
- ✅ 下载音频 - 用户下载自己创作的音乐
- ✅ 曲库管理 - 查看历史创作
- ✅ 收费系统（预留）- 0.5元/首，注册送3次免费

## 项目结构
```
music-creator/
```
